# VietTrekking

Nền tảng kết nối cộng đồng trekking Việt Nam.

## Công nghệ sử dụng

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Cài đặt

```sh
# Clone repository
git clone <YOUR_GIT_URL>

# Vào thư mục dự án
cd trekviet

# Cài đặt dependencies
npm install

# Chạy development server
npm run dev
```

## Scripts

- `npm run dev` - Chạy development server
- `npm run build` - Build production
- `npm run preview` - Preview bản build
- `npm run lint` - Kiểm tra lỗi code
